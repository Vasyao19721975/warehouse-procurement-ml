# Система оптимизации остатков товара на складе

## Описание проекта
Данный проект направлен на разработку системы, которая оптимизирует закупку товаров на складе с учетом:

- текущих остатков
- данных поставок
- популярности товаров
- сезонности

## Проблемы, решаемые системой

1. Некорректный учет остатков (невидимые товары)
2. Ошибочная оценка спроса на новые товары

## Структура проекта

- data/ — данные
- src/ — исходный код
- outputs/ — результаты

## Используемые технологии

- Python
- Pandas
- Machine Learning (в дальнейшем)

## Цель проекта

Снижение избыточных запасов и оптимизация закупок.
