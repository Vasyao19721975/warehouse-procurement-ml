# Warehouse Procurement ML

## 📌 Описание проекта
Проект для автоматизации закупок на складе с использованием анализа данных и машинного обучения.

Система:
- анализирует остатки и поставки
- оценивает скорость продаж
- прогнозирует спрос
- формирует рекомендации по закупке

---

## ⚙️ Как работает система

1. Загрузка данных (остатки, поставки)
2. Очистка и подготовка
3. Расчет продаж
4. Формирование базовых рекомендаций
5. Обучение ML модели (RandomForest)
6. Прогноз продаж
7. Финальные рекомендации

---

## 🧠 ML модель

Используется:
- RandomForestRegressor

Фичи:
- lag_1 (продажи вчера)
- lag_2 (продажи позавчера)

Метрика:
- MAE (Mean Absolute Error)
